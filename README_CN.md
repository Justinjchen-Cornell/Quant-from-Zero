## 这本书是干什么的

市面上所有的量化入门书都在教你——"学会这个策略，年化 30%。"

这本书告诉你——**那个策略的回测 +44.5% 是假的。**

不是策略失效了。是从未有效过。+44.5% 的超额收益，只是因为回测数据刚好停在 2017 年牛市顶点。把数据扩展到 2026 年，它变成了 +0.1%。

**这本书就是你亲眼看着 +44.5% 消失的全过程。**

## 11 堂课学什么

| 课 | 问什么问题 | 我知道了什么 |
|:--:|----------|------------|
| 1 | 哪个指数最危险？ | 上证 50 是 26 年综合冠军，中证 500 不如存银行 |
| 2 | PE=30 是贵还是便宜？ | 同一个数字，对不同指数含义完全相反 |
| 3 | PE 偏离正常值多远？ | 参数不通用——m=2.0 在 CSI300 有效,在深证红利完全失效 |
| 4 | 明天会跌多少？ | 不做预测，只做穷举——100,000 种未来,最坏 5% 亏 31% |
| 5 | 别人的信号能跟吗？ | 138 笔,胜率 52.9%,年化 6.7%——不如买入持有 |
| 6 | 收益像钟形曲线吗？ | 不像。创业板峰度 +11.95，极端事件是正态预测的几十倍 |
| 7 | 五连阳要不要卖？ | 在创业板上要卖，在沪深 300 上不用——看指数性格 |
| 8 | 我的回测作弊了吗？ | 离线均值比在线均值大 4.5 倍——是的，你作弊了 |
| 9 | 动量和 PE 谁赢？ | 动量更稳，PE 曾经赢过（但那是假的） |
| 10 | 哪个指数性价比最高？ | 上证 50，夏普 0.24，26 年综合冠军 |
| 11 | 涨了多少个标准差？ | 跃升等级告诉你——但即使知道了，也跑不赢买入持有 |

## 五分钟跑起来

```bash
git clone https://github.com/Justinjchen-Cornell/Quant-from-Zero.git
cd Quant-from-Zero
pip install -r requirements.txt
python code/lesson1_descriptive.py
```

不用配 API key。不用买数据。数据已经打包好了——16 个 A 股指数，1990 年到 2026 年，8683 个交易日。

## 写在最后

我不是量化研究员。我做了 16 年资产管理，见过太多人被回测骗。

写这本书，是因为市面上所有的量化入门书都在教你"怎么用这个策略赚钱"——没有一本告诉你"这个策略为什么赚不到钱"。

如果你读完这本书，以后看到任何"历史年化 XX%"的宣传，第一反应不是兴奋，而是问："数据到哪一年？训练集和测试集怎么分的？参数是调过的吗？"——那这本书就没有白写。

## 既然策略不能赚钱，这本书的价值在哪里

市面上所有的量化入门书都在教你——学会某个策略，年化30%。

这本书告诉你——**那个策略的回测 +44.5% 是假的。**

不是策略失效了。是从未有效过。+44.5% 的超额收益，只是因为回测数据刚好停在 2017 年牛市顶点。把数据扩展到 2026 年，它变成了 +0.1%。

**这本书就是你亲眼看着 +44.5% 消失的全过程。** 这比读一百篇量化论文更能教会你什么叫回测——以及为什么不要相信任何回测结果。

## 十一课内容

| # | 核心问题 | 关键发现 |
|:-:|---------|---------|
| 1 | 哪个指数风险最大？ | 上证50夏普0.24居首 |
| 2 | PE=30是贵还是便宜？ | 同一数字不同指数含义相反 |
| 3 | PE偏离正常值多远？ | 参数不具备通用性 |
| 4 | 一万种未来的分布？ | 蒙特卡洛VaR 95%=-31% |
| 5 | 外部信号可信吗？ | 138笔，不如买入持有 |
| 6 | 收益率服从正态吗？ | 不服从，峰度+11.95 |
| 7 | 连涨连跌后怎样？ | 因指数性格而异 |
| 8 | 回测作弊了吗？ | 离线均值是在线的4.5倍 |
| 9 | 动量vs价值谁赢？ | 30年尺度纯动量最稳 |
| 10 | 性价比谁最高？ | 夏普比用于排除 |
| 11 | 涨跌强度如何量化？ | 跃升等级是传感器 |
